# Codex Adapter: DevBuddy Backend

Use this profile when a Codex session needs backend-focused implementation or review.

## Routing

Select this profile for prompts involving:

- .NET, ASP.NET Core, Web API, EF Core, data models, migrations, runtime configuration, diagnostics, security, performance, or backend tests.
- API contracts, validation, persistence, authorization, observability, or deployment behavior.

## Instructions

- Keep the main DevBuddy skill as the orchestrator for business understanding, planning, approval, review, and memory updates.
- Read `skill/agents/shared/backend.md` as the role source of truth for scope, constraints, output shape, and validation expectations.
- Use installed .NET-specific skills when visible; otherwise use bundled .NET references.
- Verify relevant backend behavior when feasible.
